3D Printer — Bambu Lab P1S

Basic Operation

  • Open door when printing to avoid heat creep
  • Sparse infill pattern: Gyroid recommended
  • Enable Prime tower for multicolor prints
  • For thin objects, set outer/inner wall speed to 30 mm/s
  • Support top Z distance: 0.22 | XY distance: 0.5
  • Use Arachne wall generator for thin/rigid fonts

Tutorials & Links

  • Full Instruction Google Doc
  • Display Screen Instruction
  • AMS Installation on P1S
  • Maker World (test prints)
  • Multicolor Printing Tutorial

Weekly Maintenance

  • Clean print plate weekly with clear water
  • Bed leveling every 2 weeks via display settings
  • Clear nozzle clogs: heat to 250°C, push filament through manually
  • ⚠ Turn OFF printer before tearing down extruder

Monthly Maintenance

  • Lubricate Z-axis lead screws with SuperLube (潤滑膏)
  • Lubricate Y/Z linear rods with sewing machine oil (針車油)
  • Wipe X-axis carbon rod with isopropyl alcohol only — NO oil
  • Clean chamber fan dust in the back
  • Do NOT touch XY belt unless software flags an issue

Filament Colors Available

  • Black ×3, Matte Ivory White ×2, Bambu Green ×3
  • Purple ×2, Blue ×2, Pink ×2, Orange ×2, Red ×2
  • Gold ×3, Yellow ×2, Grey ×2, Cocoa Brown ×2
  • Turquoise ×2, Magenta ×1, Beige ×1
  • Translucent: Purple, Light Jade, Cherry Pink, Ice Blue (×2 ea)

Section 02

How to Set Up Files

Laser Cutter
Cricut Machine
3D Printer

Files for the Epilog laser cutter must be prepared in vector software (e.g. Adobe Illustrator). Lines must be clean, closed paths. Use stroke weight 0.001pt for cut lines.

File Comparison — Cut Lines

Correct
  • Single clean outline with proper closed paths
  • Holes and cutouts are well-defined, non-overlapping closed shapes
  • All lines are on correct layers (cut vs. engrave)
  • No stray points or open paths
Incorrect
  • Shapes are not properly grouped — objects overlap without being merged
  • Some circles extend outside the boundary of the main shape
  • Filled shapes mixed in with outlines — machine cannot distinguish cut vs. engrave
  • Geometry is not aligned to a unified origin point

File Comparison — Tag / Keychain Design

Correct
  • All artwork stays within the cut boundary (red outline)
  • Hole punch is a clean closed circle — properly placed inside the boundary
  • No stray paths or anchor points outside the tag border
  • Cut lines are clean, closed paths with no overlapping geometry
Incorrect
  • Stray lines and anchor points floating outside the cut boundary
  • Loose paths will be cut by the machine — wasting material
  • Design elements extend beyond the tag border
  • File not cleaned up — extra nodes likely left over from image trace
01
Use Outlines Only for Cut Lines

Set stroke to 0.001pt (hairline) with no fill. The laser reads stroke paths as cut lines.

02
Close All Paths

Every cut shape must be a fully closed path. Open paths cause incomplete cuts or skipped sections.

03
Separate Cut and Engrave Layers

Place cut lines on one layer and engrave fills/rasters on a separate layer. Use different colors to differentiate.

04
No Overlapping Shapes

Shapes that overlap will be cut twice. Use Pathfinder → Unite to merge overlapping paths into one.

Cricut Design Space reads shapes and fills directly. Use solid shapes — not outlines — for vinyl cuts. Group same-color shapes together on the same layer to minimize cuts and material use.

File Comparison — Shape vs. Outline

Correct
  • Character is built from solid filled shapes, not outlines/strokes
  • Each color group is on its own layer
  • Same-color shapes are joined (Boolean union) into single paths
  • Layers are clearly grouped and labeled by color
Incorrect
  • Design uses outlines/strokes — Cricut will cut the stroke path, not the filled shape
  • Different colors are mixed on the same layer
  • Shapes of the same color are not joined — results in unnecessary extra cuts
  • All elements are in a single group with no layer separation
01
Use Shapes, Not Outlines

Design with solid filled shapes. Remove all strokes. Cricut cuts the outline of filled regions — outlines/strokes will create unwanted double cuts.

02
One Layer Per Color

Each vinyl color should be on its own separate layer. This tells Cricut which color mat to use for each cut pass.

03
Join Same-Color Shapes

Shapes of the same color must be united (joined/welded) into a single path. This reduces cuts and avoids overlapping on the same vinyl sheet.

04
Group Elements by Color

After uniting shapes by color, group each color's shapes together. This keeps layers organized when importing into Cricut Design Space.

3D models must be properly set up in TinkerCAD or similar software before printing. Shapes must be aligned, grouped, and sized correctly — the Bambu Lab P1S cannot print floating geometry or objects too thin to extrude.

Front View

Correct
  • All shapes are grouped inside the container box
  • Face features (eyes, cheeks, nose) are properly aligned to the face
  • No shapes floating outside the main body
Incorrect
  • Container is cylindrical instead of matching the body shape
  • Face elements are misaligned — cheek circles are too far out
  • Features not centered properly on the head

Back View

Correct
  • Tail is properly embedded/grouped — not floating
  • All shapes cut together into a unified solid
Incorrect
  • Tail appears floating or not properly joined to body
  • Shapes are not merged — will create separate print islands

Side View

Correct
  • Ears sit cleanly on the head — no floating elements
  • Model sits flat on the base plane
Incorrect
  • Small shapes (nose, cheeks) are floating away from the face
  • Shapes protruding in unintended directions

Top View

Correct
  • Ears are symmetrically placed and fully attached to head
  • All elements centered on the X axis
Incorrect
  • Stray geometry visible — shapes not properly grouped
  • Elements misaligned from center axis
01
No Floating Shapes

Every shape must touch or be embedded in an adjacent shape. Floating geometry will either fail to print or fall off mid-print.

02
Shapes Must Not Protrude Out of Container

Decorative features (eyes, ears, tail) should not stick out beyond the intended container or base. Group and cut shapes together to form a unified solid.

03
Align All Features Properly

Facial features (eyes, nose, ears) must be precisely centered and aligned relative to the head/body. Check alignment from all views: front, side, top, and back.

04
Check Scale Before Export

Always verify the model dimensions in mm before slicing. An incorrect scale can result in a model that's too small for detail or too large for the print bed.

05
Avoid Thin Walls & Objects

No wall or feature should be thinner than ~0.8–1.2mm (depending on nozzle size). Objects too thin will not print — increase thickness or simplify the geometry.

06
Group and Cut Shapes Together

In TinkerCAD, use Group to merge all parts. Use holes/cutouts properly so embedded details (like recessed eyes) are correctly subtracted from the body.
