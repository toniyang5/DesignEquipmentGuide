3D Printer — Bambu Lab P1S

Basic Operation

  • Open door when printing to avoid heat creep
  • Sparse infill pattern: Gyroid recommended
  • Enable Prime tower for multicolor prints
  • For thin objects, set outer/inner wall speed to 30 mm/s
  • Support top Z distance: 0.22 | XY distance: 0.5
  • Use Arachne wall generator for thin/rigid fonts

Tutorials & Links

  • Full Instruction Google Doc
  • Display Screen Instruction
  • AMS Installation on P1S
  • Maker World (test prints)
  • Multicolor Printing Tutorial

Weekly Maintenance

  • Clean print plate weekly with clear water
  • Bed leveling every 2 weeks via display settings
  • Clear nozzle clogs: heat to 250°C, push filament through manually
  • ⚠ Turn OFF printer before tearing down extruder

Monthly Maintenance

  • Lubricate Z-axis lead screws with SuperLube (潤滑膏)
  • Lubricate Y/Z linear rods with sewing machine oil (針車油)
  • Wipe X-axis carbon rod with isopropyl alcohol only — NO oil
  • Clean chamber fan dust in the back
  • Do NOT touch XY belt unless software flags an issue

Filament Colors Available

  • Black ×3, Matte Ivory White ×2, Bambu Green ×3
  • Purple ×2, Blue ×2, Pink ×2, Orange ×2, Red ×2
  • Gold ×3, Yellow ×2, Grey ×2, Cocoa Brown ×2
  • Turquoise ×2, Magenta ×1, Beige ×1
  • Translucent: Purple, Light Jade, Cherry Pink, Ice Blue (×2 ea)

xTool F1 Ultra Laser Cutter

Overview

  • High-speed, dual-laser: 20W Fiber + 20W Diode.
  • Supports engraving, and cutting on various materials
  • Uses xTool Creative Space (XCS) software for design and machine control

Importing Files

  • Export your design from Illustrator as SVG, DXF, PNG, or JPG
  • Open xTool Creative Space (XCS) on your computer
  • Import the exported file using the "Image Tool"
  • Arrange and adjust your design before sending it to the laser

Material Settings

  • Use the xTool EasySet material library for recommended settings: EasySet Library
  • Always run a test array on new materials before full projects
  • Color engraving is supported — check tutorials for settings

Section 02

How to Set Up Files

Laser Cutter
Cricut Machine
3D Printer

Prepare files in Adobe Illustrator for laser cutting materials like cardboard, wood, or acrylic.

01
Cut Lines

Use red (R:255, G:0, B:0) with a 0.1 pt stroke outline for cut lines.

02
Engrave

Use blue (R:0, G:0, B:255) or black color shapes or simple images with one or two colors for engraving.

03
Close All Paths

All paths should be closed, with no unnecessary paths or shapes.

04
Join Overlapping Shapes

Join overlapping shapes using the Shape Builder or Pathfinder tool.

05
Embed Images & Outline Text

All images must be embedded, and all text must be converted to shapes using the Create Outlines function.

!

Export as an Illustrator or SVG file, and make sure to check scale before exporting.

File Comparison — Cheese Shape Design

Correct
  • Holes and cutouts are properly cut with pathfinder tool
  • Paths are properly closed with clean shapes
  • Cut lines use red (R:255, G:0, B:0) with a 0.1 pt stroke and no fill
Incorrect
  • Holes and cutouts are not properly cut
  • Paths are open, with overlapping lines and shapes
  • Cut elements should be outlines only, with no filled shapes

File Comparison — Pikachu Keychain Design

Correct
  • Scale is correct
  • Colors for both cut and engraving shapes are correct
  • Paths are properly closed with clean shapes
  • All elements are properly placed and aligned
  • Text is converted to shapes and images are embedded
Incorrect
  • Scale is incorrect
  • Cut lines should be red (R:255, G:0, B:0) with a 0.1 pt stroke
  • Unnecessary paths should be deleted
  • The circle is too close to the edge, text is not properly aligned
  • Text must be converted to shapes and images must be embedded

Prepare designs in Adobe Illustrator for vinyl cutting or crafting with a Cricut machine.

01
Use Shapes for Cuts

Use shapes to indicate cut areas; do not use outlines.

02
Join Overlapping Shapes

Join overlapping shapes using the Shape Builder or Pathfinder tool to avoid unnecessary extra cuts.

03
Limit and Group Colors

Use no more than five colors, must group shapes of the same color together.

04
No Pixel Images

All images must be vectorized using Image Trace, limit color palette to under 5 colors when image tracing.

05
Outline All Text

Convert all text to shapes using the Create Outlines function.

!

Export as an SVG file, and make sure to check scale before exporting.

Tutorial — How to Fix Your File

Files must be properly set up in TinkerCAD or similar 3D modeling software before printing.

01
No Floating Shapes

There should be no floating shapes that are not connected to other shapes./p>

02
Align All Elements

Ensure all shapes are properly aligned.

03
Check Scale

Check that the scale is correct using the ruler tool.

04
Avoid Thin Walls

Avoid thin walls and fragile objects.

05
Combine Shapes Properly

Combine shapes and holes appropriately by clicking on Union Group.

!

Export as an STL file, and make sure to check scale before exporting.

Front View

Correct
  • The scale of all shapes is correct
  • Shapes are properly aligned and symmetrical
  • The head is properly placed and connected to the body
Incorrect
  • The scale of some shapes is incorrect (ears, eyes, legs)
  • Shapes are not properly aligned
  • The head is not properly placed and is barely touching the body

Back View

Correct
  • The legs and tail are properly sized and aligned
  • The body (box) has beveled edges
Incorrect
  • The legs and tail are misaligned and incorrectly sized
  • The body edges are sharp and not beveled

Side View

Correct
  • There are no floating shapes; all shapes are properly connected
  • The legs are aligned with the edge of the body
Incorrect
  • Floating shapes are present (eyes, nose, and tail)
  • The legs protrude from the body and are not aligned

Top View

Correct
  • The hole in the body (box) is clean and centered
  • The legs do not protrude into the interior of the body (box)
Incorrect
  • The hole in the body (box) is off-center, resulting in uneven wall thickness
  • The legs protrude into the interior of the body (box), indicating that shapes are not properly grouped
